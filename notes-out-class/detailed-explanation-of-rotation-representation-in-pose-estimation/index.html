我们不需要知道四元数在四维空间中是如何旋转的。

定义

一个四元数$\mathbf{q}$可以被定义为如下形式，相互等价：

其中：

  • $\mathbf{q}_v = \mathbf{i}q_x + \mathbf{j}q_y + \mathbf{k}q_z = (q_x, q_y, q_z)$
  • $\mathbf{i}^2 = \mathbf{j}^2 = \mathbf{k}^2 = \mathbf{i}\mathbf{j}\mathbf{k} = -1$
  • $\mathbf{i}\mathbf{j} = -\mathbf{j}\mathbf{i} = \mathbf{k}$
  • $\mathbf{j}\mathbf{k} = -\mathbf{k}\mathbf{j} = \mathbf{i}$
  • $\mathbf{k}\mathbf{i} = -\mathbf{i}\mathbf{k} = \mathbf{j}$

我们称$q_w$为四元数$\mathbf{q}$的实部，$\mathbf{q}_v$为虚部，$\mathbf{i}$，$\mathbf{j}$和$\mathbf{k}$是虚数单位。

注意，虚数单位之间的乘法不遵守交换律。

对于虚部$\mathbf{q}_v$，我们可以施加所有普通向量运算操作，例如，加法，缩放，点乘，叉积等。
