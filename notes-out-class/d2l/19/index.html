19 卷积层

从全连接到卷积

分类猫和狗的图片

  • 使用一个还不错的相机采集图片（12M像素）
  • RGB图片有36M元素
  • 使用100大小的单隐藏层MLP，模型有3.6B元素
    • 远多于世界上所有猫和狗总数（900M狗，600M猫）

回顾：单隐藏层MLP

有两个原则：

  • 平移不变性

    在图片中的任何一处地方对同一个物体进行识别时，识别器应该是不变的。

  • 局部性

    识别某物体不需要全局的信息，只需要关注局部即可。

重新考察全连接层

我们将全连接层的输入和输出变形为矩阵（宽度，高度），然后将权重由$(\mathrm{h}, \mathrm{w})$变到4-D张量$(\mathrm{h}^\prime, \mathrm{w}^\prime)$

其中：V是W的重新索引$v_{i, j, a, b} = w_{i, j, i + a, j + b}$

注意，这里不能用矩阵&向量的思维去尝试理解上面的式子，因为人是无法想象4D空间的，最好是通过下标对应的方式去理解。

  • 首先解释$h_{i, j} = \sum_{k, l}w_{i, j, k, l}x_{k, l} = \sum_{a, b}v_{i, j, a, b}x_{i + a, j + b}$：

    首先，$h_{i, j}$是输出矩阵中的一个元素，不是整个输出矩阵。

    在4D权重$w_{i, j, k, l}$中，$i, j$代表输出的元素在输出矩阵中的位置，$k, l$代表输入的元素在输入矩阵中的位置。

    那么在$\sum_{k, l}w_{i, j, k, l}x_{k, l}$中，相当于是遍历了输入矩阵的每一个元素，然后计算这一整个输入矩阵在对应4D权重影响下的输出。

    回忆实际的卷积（假设你已经了解过卷积相关的概念），如果对于4D权重$w_{i, j, k, l}$的每一个$i, j$组合，其权值矩阵的值都相同，那么就可以对这个权重进行降维（因为下标$i, j$就没有用了），从而得到我们实际用的卷积。

  • 然后解释一下后面的换下标，即$v_{i, j, a, b} = w_{i, j, i + a, j + b}$：

    当对$w$的下标进行修改时，计算会发生如下的变化：

    然后再令$v_{i, j, a, b} = w_{i, j, i + a, j + b}$，就能够得到$h_{i, j} = \sum_{a, b}v_{i, j, a, b}x_{i + a, j + b}$。

原则1 - 平移不变性

  • x的平移导致h的平移$h_{i, j} = \sum_{a, b}v_{i, j, a, b}x_{i + a, j + b}$

  • v不应该依赖于$(\mathrm{i}, \mathrm{j})$

  • 解决方案：$v_{i, j, a, b} = v_{a, b}$（卷积核的权值共享）（压缩掉前两维）

    这就是2维卷积（严格来说叫交叉相关）

原则2 - 局部性

  • 当评估$h_{i, j}$时，我们不应该用远离$x_{i, j}$的参数

  • 解决方案：当$\left|a\right|, \left|b\right| > \Delta$时，使得$v_{a, b} = 0$

    这就是卷积核的大小

总结

对全连接层使用平移不变性和局部性得到卷积层

卷积层

二维交叉相关

  • 平移不变性：只有一个卷积核
  • 局部性：卷积核只覆盖了图片的一部分

二维卷积层

  • 输入$\mathbf{X}$：$n_h \times n_w$

  • 核$\mathbf{W}$：$k_h \times k_w$

  • 偏差$b \in \mathbb{R}$

  • 输出$\mathbf{Y}$：$(n_h - k_h + 1) \times (n_w - k_w + 1)$

  • $\mathbf{W}$和$b$是可学习的参数
